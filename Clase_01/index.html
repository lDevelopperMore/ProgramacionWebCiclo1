¿Que significa html?

html significa hyper text markup language lenguaje de marcado de hipertexto Hypertext hipertexto es la capacidad de saltar de una pagina a otra mediante links vinculos Markup Marcado significa que no es un lenguaje de programacion puro como js y demas. sino un lenguaje para usar etiquetas para marcar el contenido ejemplo etiquetar un titulo con h1 o un parrafo con p lenguaje es el sistema de reglas que permite al navegador leer entiendan lo mismo

¿Que versiones existen?

Versiones HTML 1 1991 Solo texto y enlaces. HTML 2 1995 Nacen los formularios. HTML 3.2 1997 Tablas y fotos. HTML 4.01 1999 Llega el diseño con CSS. HTML5 2014 Multimedia, Apps y Semántica.

¿Quien invento html?

Sir Tim Berners-Lee o hizo en 1989 (aunque la primera versión oficial salió en 1991) mientras trabajaba en el CERN (la Organización Europea para la Investigación Nuclear) en Suiza. Tim Berners-Lee no solo creó el HTML, sino que inventó todo lo que hoy conocemos como la World Wide Web (WWW). Su problema era que los científicos del CERN tenían mucha información esparcida en diferentes computadoras y no había una forma fácil de compartirla. Para solucionar esto, él creó tres tecnologías fundamentales que aún usamos hoy: HTML: El lenguaje para escribir las páginas. HTTP: El protocolo para enviar esas páginas de una PC a otra. URL: La dirección para encontrar esas páginas.

Dato curioso 1 ¿Sabias que?

A diferencia de otros inventores que se hicieron multimillonarios cobrando por sus inventos, Tim Berners-Lee decidió que el HTML y la Web debían ser gratis para todo el mundo.

Dato curioso 2 ¿Quién manda hoy en el HTML?

Aunque Tim lo inventó, él ya no decide solo los cambios. Ahora existe una organización llamada W3C (World Wide Web Consortium), la cual él todavía dirige. Ellos son los que se ponen de acuerdo para decidir, por ejemplo, qué etiquetas nuevas se agregan a HTML5

Tarea ¿Que son las etiquetas vacias? y ¿Que son las etiquetas semanticas?

Etiquetas vacias

Son las etiquetas que dictan una accion o insertan algo y no etiquetan algo como ejemplo br, img, hr

Etiquetas semanticas

Son las etiquetas que etiquetan y categorizan datos son la base escencial de apertura del lenguaje de marcado para la arquitectura del mismo ejemplo nav, main footer
