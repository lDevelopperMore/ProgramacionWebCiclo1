¿Que son las tablas y las listas?

1. Tablas :(table)

Sirven para organizar datos en filas y columnas. Es como una cuadrícula de Excel.

1.1 Componentes de las tablas

Etiquetas Definicion
td Table cells Son los cuadros individuales. Se dividen en th (títulos) y td (datos).
tr Table row crea una fila horizontal
th Table header celda de titulo"resalta el texto"

2. Listas: (li)

Permiten agrupar elementos relacionados.

2.1 Clases de listas

Desordenadas (ul)

  • TD
  • TR
  • TH

Ordenadas (ol)

  1. TD
  2. TR
  3. TH

2.2 Descripcion de listas

Describen lo que hay dentro de las listas
