Ejemplos practicos

A. Imagen con Texto Alternativo (SEO)

Las imágenes deben llevar siempre el atributo alt para accesibilidad. El atributo alternativo El atributo requerido proporciona un texto alternativo para una imagen, si el usuario para Alguna razón no puede verlo (por la conexión lenta, un error en el SRC o si el usuario usa un lector de pantalla).alt El valor del atributo debe describir la imagen:alt

Si un navegador no puede encontrar una imagen, mostrará el valor del atributo:alt

Tamaño de la imagen - Ancho y altura

Puedes usar el atributo para especificar el ancho y altura de una imagen.style

style="width:500px;height:600px;" Los atributos y siempre definen el ancho y la altura de los atributos Imagen en píxeles.width height

¿Anchura y altura, o estilo?

Los atributos en el head son validos pero no son recomendables , , y son todo válido en HTML.widthheight style Sin embargo, sugerimos usar ese atributo. Evita que las hojas de estilos cambien El tamaño de las imágenes:style

Imágenes en otra carpeta

si tienes tus imágenes en una subcarpeta, debes incluir la carpeta nombre en el atributo:src

img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Imágenes en otro servidor/sitio web

lgunas páginas web apuntan a una imagen en otro servidor.Para apuntar a una imagen en otro servidor, debes especificar un absoluto (completo) URL en el atributo:src

img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com"

Imágenes animadas

HTML permite GIFs animados:

img src="programming.gif" alt="Computer Man" style="width:48px;height:48px;

Imagen con enlace

Imagen como enlace Para usar una imagen como enlace, coloca la etiqueta dentro de la etiqueta:

a href="default.asp"> img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">/a

Imagen flotante

Utiliza la propiedad CSS para que la imagen flote hacia la derecha o la izquierda de un texto:float

img src="smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px;"> The image will float to the right of the text.

img src="smiley.gif" alt="Smiley face" style="float:left;width:42px;height:42px;">The image will float to the left of the text.
